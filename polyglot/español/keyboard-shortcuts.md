When studying Spanish, the need for key-board shortcuts arises.  <br>

        Typing Accents
        á = Alt + 0225
        Á = Alt + 0193
        é = Alt + 0233
        É = Alt + 0201
        í = Alt + 0237
        Í = Alt + 0205
        ó = Alt + 0243
        Ó = Alt + 0211
        ú = Alt + 0250
        Ú = Alt + 0218
        ñ = Alt + 0241
        Ñ = Alt + 0209
        ü = Alt + 0252
        Ü = Alt + 0220

        Punctuation marks:
        ¡ = Alt + 0161
        ¿ Alt + 0191
        « Alt + 0171
        » Alt + 0187
